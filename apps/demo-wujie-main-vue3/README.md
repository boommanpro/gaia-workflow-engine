# Vue3 微前端工作流管理项目

这是一个基于 Vue3 的微前端主应用，用于管理工作流的增删改查，并与 frelayout 微应用进行交互。

## 功能特性

- 工作流的增删改查
- 本地存储模拟数据库
- 与 frelayout 微应用集成
- 微前端架构 (使用 wujie)
- 响应式界面设计

## 目录结构

```
src/
├── components/        # 公共组件
├── views/             # 页面视图
├── router/            # 路由配置
├── store/             # 状态管理
├── utils/             # 工具函数
├── styles.css         # 全局样式
└── main.js            # 入口文件
```

## 安装和运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 微前端集成

本项目作为主应用，通过 wujie 微前端框架与 frelayout 编辑器进行集成：

1. 工作流列表和管理功能在主应用中实现
2. 工作流编辑功能在 frelayout 微应用中实现
3. 通过 wujie 的通信机制进行数据交换

## 数据存储

使用 localStorage 模拟数据库存储工作流数据：

- 数据存储在 `workflow-db` 键下
- 包含工作流的基本信息和内容数据
- 支持增删改查操作

## 跨应用通信

通过 wujie 提供的通信机制实现主应用与微应用的数据交换：

1. 主应用向微应用传递工作流数据
2. 微应用向主应用发送保存请求
3. 通过事件总线进行实时通信
