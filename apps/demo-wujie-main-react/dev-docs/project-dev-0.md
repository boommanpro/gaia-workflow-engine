## 项目介绍

当前项目是工作流的示例管理系统，用于管理工作流的配置，版本等信息，其中工作流的采用wujie打通。

## 项目主体介绍
1. 前端采用react19框架，ui采用semi-design(https://semi.design/zh-CN/)。
2. 工作流前端采用wujie打通项目demo-free-layout，详情可以参考vue3版本的打通：demo-wujie-main-vue3。wujie的官方网址是：https://github.com/Tencent/wujie

## 工作流示例管理系统功能
1. 工作流的卡片式管理，具备增加，修改，版本切换，删除等功能。
2. 可以通过点击工作流卡片查看工作流的具体日志，执行日志和版本强相关。
3. 工作流的创建具备模板功能，后端会提供模板列表，当基于模板创建时，采用后端默认的数据。
